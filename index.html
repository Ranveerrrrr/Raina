👋 Welcome to Raina

This is a school project coded and hosted by Ranveer.
Explore tools for phishing awareness, cybersecurity, and OSINT basics.

🛡️ In today’s digital world, security is more important than ever. Hackers often exploit weak passwords, phishing links, and exposed subdomains to gain access to sensitive information.

This project is designed to raise awareness about cybersecurity by providing tools that show how attackers think — and how you can protect yourself. From checking if a site is safe, to finding hidden subdomains, every tool here teaches a small but powerful concept in online safety.

🗂️ File / Link Malware Scan

Scan a downloadable file link using VirusTotal to check if it’s malicious.

🌐 SubDomain Finder

Find all SubDomain's a company has Made.

🔗 URL Safety Check

Verify if a website is flagged as unsafe by Google Safe Browsing.

🔑 Password Leak Checker

Check if your password appeared in known data breaches.

🎣 Phishing Awareness

Learn how phishing works and try an interactive phishing simulation.

💻 Device Info

See what details your browser leaks — user agent, platform, and more.

📧 Email Breach Checker

Check if your email was involved in any public data breach.

🔐 Hash Generator

Generate MD5, SHA-1, and SHA-256 hashes from your input.

🌍 IP Lookup

Check IP address details like location, ISP, and geodata.

🔍 Link Analyzer

Get full redirection paths, shortened links, and headers.

🧠 Password Strength

See how strong your password is and get suggestions.

📷 QR Code Decoder

Upload a QR image and extract the encoded URL or message.
